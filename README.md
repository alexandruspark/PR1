# PR1
Inventory manager
Program in C that reads inventory data from a file, detects which equipment is below the minimum required, calculates how much it would cost to replace it, and displays a summary.
Example:
LOAD DATA FROM FILE. ENTER FILE NAME >> equip1.txt MINIMUM STOCK NOTICES (1-DESKTOP, 2-LAPTOP, 3-PRINTER, 4-MONITOR, 5-KEYBOARD, 6-MOUSE) EQUIPMENT - REQUIRED - RESTOCK COST [EUR] 3 - 2 - 350.00 3
5 - 5 - 137.00 1 - 1 - 775.15 TOTAL RESTOCK COST [EUR]: 1262.15
